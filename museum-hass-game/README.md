# 7 HASS: Skill Quest

A browser-based revision game for Year 7 Humanities and Social Sciences (History).

## Instructions
1. Upload the entire folder to a GitHub repository.
2. Ensure `index.html` is at the root.
3. Use GitHub Pages to host the site.
4. Players assume the role of a junior researcher investigating three First Nations history exhibits.

## Technical Details
- Built with HTML5, CSS3, and Vanilla JavaScript.
- Data-driven via JSON.
- Responsive to a 16:9 aspect ratio (fixed 1280x720 container).
