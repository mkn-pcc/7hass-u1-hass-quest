# 7 HASS: Skill Quest

A browser-based revision game for Year 7 History.

## Folder Structure
- `/assets/backgrounds/`
- `/assets/portraits/`
- `/assets/ui/`
- `/assets/sources/`
- `/data/` (Must contain cases.json, dialogue.json, sources.json)

## Gameplay
Players analyze sources to identify:
1. Source Type (Primary/Secondary)
2. HASS Category (Social, Environmental, Cultural)
3. Interpretation of evidence.

## Deployment
Upload to GitHub and enable GitHub Pages on the root directory.
